Despite the improvements, the system still faces issues with:- 

📉 Build Recommendation Accuracy
- Item recommendations and build order are not yet fully reliable.

🧩 Root Cause
- Basic items and advanced (full) items are currently treated at the same level
- This lack of hierarchy leads to incorrect build path predictions

      - Ensure the application is accessible and practical during live gameplay
